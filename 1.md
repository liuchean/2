## 1. GNU 的 GPL 與 Opensource 開放原始碼授權
GNU 的 GPL 授權主要強調自由的學習，但並不是說免費

### 有底下的權力與義務：
1. **取得與執行**：可依需求運行軟體。
2. **複製**：可自由拷貝散佈。
3. **修改 (Memory)**：可根據需求改寫原始碼。
4. **再發行**：允許分享修改後的版本。
5. **回饋社群 **：鼓勵將修改後的成果分享回饋。

### 關鍵限制
> [!NOTE]
> * **不可修改授權**：不能取消 GPL 授權，修改後仍須維持 GPL。
> * **不可單純販賣**：禁止在未增加額外價值的情況下單純出售軟體。

## 2. 開放原始碼 (Open Source)
1998 年由 OSI 提出，旨在消除商業公司對 "Free" 字眼的疑慮，強調開發模式的相容性。
### 核心定義 (OSI 標準)
1.  **公佈原始碼**：用戶具修改權與編譯權。
2.  **任意再散佈**：允許販售或作為其他軟體元件，作者不可收取額外費用。
3.  **允許衍生作品**：須允許修改，並可使用相似授權發表。
4.  **識別性**：可用不同名稱或編號散佈。

### 八大「不限制」原則
* **對象**：不可限制某些個人或團體使用。
* **領域**：不可限制商業或學術等特定領域應用。
* **產品**：不可限制必須在某些特定產品內使用。
* **條款**：不可具備排他性條款（例如禁止用於教育研究）。

## 3. linux 基本指令

### 3.1 ls 與 ll
一般用戶的身份登入 Linux 系統，同時啟動一個終端機在桌面上。現在讓我們來執行兩個指令，確認一下如何操作系統與觀察輸出的資料。
* **`ls` (List)**：
    * **功能**：單純列出檔案或目錄的**名稱**。
    * **輸出範例**：`Desktop Documents Downloads...`
    * **適用情境**：只想快速確認有哪些檔案存在時。

* **`ll` (ls -l 的別名)**：
    * **功能**：顯示檔案的**詳細資訊**（Long format）。
    * **包含內容**：
        * 檔案權限（Permissions）
        * 檔案容量（Size）
        * 修改時間（Timestamp）
        * 擁有者與群組資訊
    * **適用情境**：需要查閱權限、確認檔案大小或更新時間時。

### 3.2 歷史命令功能

在 Linux 文字界面中，若要重複使用或修改曾下達過的指令，可透過以下方式：
#### 1. 方向鍵與快速移動
* **上 / 下鍵**：在指令紀錄中切換，呼叫出之前執行過的指令。
* **左 / 右鍵**：在同一行指令中左右移動游標以進行修改。
* **Home / End 鍵**：游標快速跳轉至該行指令的最前面或最後面，提升編輯效率。

#### 2. history (歷史命令)
當需要查找較久之前的指令紀錄時，可使用 `history` 指令：
* **功能**：列出系統紀錄中所有執行過的指令清單。
* **適用情境**：忘記長指令的參數，或需要查閱過去的操作流程時。


### 3.3 離開系統與關機建議

在 Linux 中，「離開系統」通常是指關閉特定的軟體環境（如終端機或圖形界面），而非直接切斷電源。

#### 1. 關閉終端機 (Terminal)
雖然可以直接點選視窗右上角的「X」，但為了確保背景任務正常結束，建議使用以下正確方式：
* **輸入指令**：執行 `exit` 或 `logout`。
* **快捷鍵**：在未輸入任何文字的情況下，按下 `[Ctrl]` + `d`。

#### 2. 登出圖形界面
* **建議進行的步驟**：
    1. 點選「概覽 (Activities)」檢查所有桌面。
    2. 確認所有開啟的軟體（如瀏覽器、文件編輯器）皆已關閉。
    3. 執行「登出 (Logout)」。

#### 3. 關機
關機的指令只要有四個`poweroff`，`halt`，`shutdown -h now`，`systemctl poweroff`，上述的任何一個指令均可關機。但無論使用哪個指令關機，其實最終都是呼叫最後一個，亦即『 systemctl poweroff 』進行關機的行為。

> [!NOTE]
> * **本地登入 (tty1~tty6)**：不論是系統管理員 (root) 或一般帳號，只要是在本機前面操作，通常都可以直接下達 `poweroff` 關機。
> * **遠端連線 (SSH/網路)**：
>     * **一般帳號**：無法透過網路遠端關閉系統。
>     * **管理員帳號**：僅有管理員具備透過網路連線關機的權限。
